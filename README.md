# improving-my-styles

Set of components used to learn css

## Commands

### Development

- Run webpack mode development

  ```
  npm run start
  ```

- Run ESlint

  ```
  npm run lint
  ```

- Fix ESlint

  ```
  npm run lint:fix
  ```

- Format with Prettier

  ```
  npm run format
  ```

### Production

- Run webpack mode production

  ```
  npm run build
  ```

- Predeploy for Github pages

  ```
  npm run predeploy
  ```

- Deploy in Github pages

  ```
  npm run deploy
  ```
